Learning Outcome Alignment - Demo Tool

This tool enables a user to view relevant entry-level job postings, then parse and process those postings -- in real time. Finally, a set a learning outcomes is created for said posting and compared with a pre-generated set of outcomes directly sourced from UNT cybersecurity assessments. Get started with the demo by pressing the orange button below!

NOTE: This demo is a work in-progress.
